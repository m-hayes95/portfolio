Game Pitch

Make quick decisions to outwit your A.I opponent and defeat them in turn-based rock, paper, scissors, and grid combat! Dance to the beat on the disco floor!

Gameplay Video

Development Details

All assets except for the hearts and win, loss, draw banners, music and sounds were created by myself using the Unity Engine and Figma.

UX / UI

One of the things I wanted to focus on for this project was UI. Therefore, I decided to design cards using bright colours to explain how the game works. 

This was done to try and combat the major issue of play testers not understanding the mechanics and how they work.

An improvement that can be made with the cards is to reduce the amount of text per card to make it more ​​digestible by the reader and make it less likely for the player to skip.

Project Goals

  • Create a AI enemy that responds to the player
  • Use random generation as a core component to make the game more replayable
  • Focus on UI to improve the look of the game

Project Thoughts

The enemy AI is currently too primitive and does not react to the player's moves allowing the player to easily win everytime when planning or losing to the luck of the random generation.​ Perhaps having a more complex UI such as a behaviour tree for Enemy AI would be more beneficial as the AI will be able to select an appropriate action depending on the player's previous action.

Whilst playtesting, I found that players did not understand how the game works well. Cards were either not looked at or it seemed the cards had too much text for players too easily digest. I would like to add a tutorial to explain through playing how the game rules work and re-work the card text to be fewer words to make it more readable.

The game also has some bugs that need to be fixed, the major bug being where the AI will not make any moves and stay in the same place.
